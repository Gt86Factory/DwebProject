Qu'est ce que le Développement Web

Un développeur web est un informaticien spécialisé dans la programmation ou
expressément impliqué dans le développement des applications du World Wide Web,
ou des applications qui sont exécutées à partir d'un serveur web sur un navigateur
web et qui utilisent le protocole HTTP comme vecteur de transmission de l'information.
Le développeur web effectue la réalisation technique et le développement informatique
d'un site web. A l'aide du cahier des charges établi par le chef de projet, le développeur
web programme les fonctionnalités qui correspondent aux besoins du client pour son site internet.
Lors de la phase de conception, le développeur analyse le projet à partir des éléments
techniques, consignés dans un cahier des charges. La partie programmation consiste ensuite
en de l'encodage informatique (rédaction de lignes de code).

Qu'est qu'un develôppeur web?

Son but est de développer des interfaces web adaptées pour son client.
Il peut s'agir de sites internet, d'applications web ou d'outils en ligne.
Il répond à un cahier des charges élaboré par un chef de projet.
Il contribue également à sa rédaction, en conseillant sur les aspects techniques que ne maîtrise pas forcément le chef de projet.
Une partie des développeurs web sont des développeurs informatiques qui se sont redirigés vers le web, avec l'émergence d'Internet.

Débouchés et poursuites d'études

Une fois le titre professionnel développeur web et web mobile obtenu, si vous le souhaiter vous pouvez poursuivre vos études ou entrer sur le marché de l'emploi.

Si vous souhaiter compléter votre formation avec un autre titre professionnel afin de multiplier vos compétences dans le domaine informatique :

- Titre Professionnel Concepteur Développeur d’Application (CDA)
-Titre professionnel Ingénieur en système d’information


Si vous souhaiter poursuivre vos études vous pourrez vous dirigez vers un bac +3 :

- Licence informatique
- Licence professionnelle développeur web et multimédia
- Licence professionnelle métiers de l’informatique : applications web parcours développeur full stack

Si vous entrer sur le marché de l'emploi vous pourrez évoluer vers ces métiers :

- Architecte d'application - Responsable informatique dans une PME - Ingénieur d'étude informatique
- Expert informatique (méthodologie, qualité,...) - Chef de projet informatique - Consultant informatique
- Développeur web - Développeur web front-end - Développeur web back-end - Développeur web fullstack
- Développeur web mobile - Informaticien de développement - Études et développement informatique

Vous pourrez travailler dans ces secteurs d'activités :

- Entreprise de Services Numériques (ESN ou ex SSII) réalisant des prestations de développement d'applications
- En régie ou au forfait structure utilisatrice - De type entreprise du secteur privé ou public, possédant un service
dédié aux études et aux développements informatiques - Agence web - Activité d'informaticien d'études indépendant
- Architecture web - Conduite projet multimédia - Programmation.
